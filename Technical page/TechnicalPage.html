    • ❖Colour Gradient

    In this section i will be showing you how to use colour gradients which will allow you to style your page more effectively.

    Firstly, you will need to create the element you want to style, in my case it will be ".box1", Afterwards you will need to set the following property to your element.

    .Background : Linear-Gradient ( 90deg, Colour1, Colour2)

    In my case i used the following properties to add a colour gradient to ".box1" (example shown below).

    Background : Linear-Gradient ( 90deg, #69f44b ,#824bf4 )

    You can also apply any angle you like simply by changing the angle entered, for example, I will be changing from 90deg to 180deg (example shown below).

    Background : Linear-Gradient ( 180deg, #69f44b ,#824bf4 )

    Furthermore, it does not have to be 2 colours as you can input as many colours as you like to achieve the best possible result, for example, I will be creating a colour gradient with 3 colours (example shown below).

    Background : Linear-Gradient ( 90deg, #69f44b ,#4bbef4 ,#824bf4 )
